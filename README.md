## English Version

### Overview
Random ID Generator allows you to create realistic and culturally adapted identities for 10 countries including France, USA, Japan, UK, Germany, Australia, Canada, Spain, Italy, and Brazil. Each profile can include:

- First Name and Last Name
- Gender
- Age
- Pseudo / Username (automatic)
- Address (Street, Number, City)
- Optional fields: religion, hobbies, pet, etc.
- Random Avatar

You can generate multiple identities at once and export the results in **JSON**.

### Features
- Multi-country support (10 countries)
- Large dataset of names, streets, and cities
- Realistic but fictive addresses
- Multi-generation of IDs
- Automatic pseudo and avatar
- JSON export for integration into projects

### Usage
1. Select the country.
2. Set number of IDs to generate.
3. Click "Generate".
4. Copy or export results in JSON.

### Tech
- HTML / CSS / JS
- No backend required
- Fully client-side, lightweight, fast
